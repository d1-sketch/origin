# BlitzOrigins

BlitzOrigins is the official website for **BlitzOrigins**, a digital marketing agency focused on helping brands grow through strategy, creativity, and data-driven execution.

This repository contains the source code for the BlitzOrigins public-facing website.

## About BlitzOrigins

BlitzOrigins delivers modern digital marketing solutions designed to elevate brand visibility, engagement, and conversions.  
Our approach combines creative storytelling with analytics-backed strategies to drive measurable results.

## Website Features

- Modern, premium UI/UX design  
- Fully responsive across all devices  
- Smooth animations and transitions  
- Sticky navigation with scroll-based effects  
- Services showcase section  
- Contact form with email integration  
- Lightweight, fast-loading single-page architecture  

## Tech Stack

- HTML5  
- CSS3 (custom properties, gradients, animations)  
- Vanilla JavaScript  
- Google Fonts (Playfair Display, Montserrat)  

## Project Structure


The site is intentionally kept simple and efficient, with all functionality contained in a single file for ease of deployment and maintenance.

## Deployment

This website can be deployed on any static hosting platform, including:

- GitHub Pages  
- Netlify  
- Vercel  
- Traditional web hosting  

Simply upload or serve the `index.html` file.

## Contact Form

The contact form uses a mailto-based submission method:

- Opens the user’s default email client  
- Pre-fills inquiry details  
- No backend or server required  

This approach prioritizes simplicity and security. The form can be upgraded to a backend solution if needed.

## Customization

- Brand colors can be adjusted via CSS variables in `:root`  
- Content can be edited directly within the HTML  
- New services can be added by duplicating service card components  
- Animations and transitions are fully customizable
